Project Saya

Website Toko Online Roti

Proyek ini merupakan desain dan implementasi website toko roti berbasis web dengan antarmuka yang menarik dan responsif. Website menampilkan berbagai menu navigasi seperti Home, Produk, Tentang Kami, dan Manual Aplikasi. Di bagian atas terdapat informasi kontak, alamat email, serta identitas toko. Tampilan halaman utama dilengkapi ilustrasi kue dan cupcake dengan dekorasi lilin, menonjolkan kesan manis dan elegan untuk menarik perhatian pengunjung.

Website Kasir

Aplikasi desktop ini berfungsi sebagai sistem kasir sederhana untuk transaksi penjualan. Fitur yang tersedia meliputi input data barang (nama, harga, dan kuantitas), perhitungan otomatis subtotal dan total harga, serta proses pembayaran dengan kalkulasi kembalian. Aplikasi dilengkapi tombol INPUT, BAYAR, RESET, dan EXIT untuk mempermudah operasional. Antarmuka didesain dengan warna mencolok untuk memudahkan identifikasi elemen penting.

Aplikasi Toko Sepatu

Aplikasi desktop berbasis VB ini dirancang untuk mengelola operasional toko sepatu. Halaman utama menampilkan ilustrasi toko sepatu dengan navigasi berbentuk tombol di sisi kanan, yang mengarah ke menu Pembayaran, Barang, Kasir, dan Keluar. Antarmuka dibuat sederhana dan intuitif, memudahkan pengguna dalam mengakses fitur sesuai kebutuhan manajemen toko.
